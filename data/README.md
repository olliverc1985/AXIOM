# AXIOM Data Files

## Training Data

- **stsbenchmark.tsv** — STS Benchmark dataset (8,628 sentence pairs with similarity scores 0-5). Used for Stage 1 pretraining of the semantic encoder. Source: [STS Benchmark](https://ixa2.si.ehu.eus/stswiki/index.php/STSbenchmark).

- **fusion_training_corpus.json** — Tier-labelled queries for classification training. Each entry has `text`, `tier` (Surface/Reasoning/Deep), and optional `alt_tier`. Used for Stage 2 fine-tuning.

- **hard_negatives.json** — Contrastive training pairs for STS pretraining. Optional, improves encoder quality.

## Evaluation Data

- **eval_set1_synthetic.json** — 334 synthetic queries across diverse categories with ground-truth tier labels.
- **eval_set2_adversarial.json** — 333 adversarial queries designed to confuse complexity classification.
- **eval_set3_realworld.json** — 333 real-world queries from diverse domains.

Combined, these form the 1,000-query diverse evaluation set.

## RouterBench Data (not tracked)

The `routerbench/` directory contains RouterBench benchmark data (Hu et al., ICML 2024). These files are large and not tracked in git:

- **routerbench_5shot.pkl** — Raw RouterBench pickle (163MB). Download from [HuggingFace](https://huggingface.co/datasets/withmartian/routerbench).
- **routerbench.jsonl** — Preprocessed JSONL (122MB, 36,511 records). Generated by `scripts/preprocess_routerbench.py`.

To regenerate:
```bash
pip install pandas
python scripts/preprocess_routerbench.py
```
